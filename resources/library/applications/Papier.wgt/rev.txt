Version 2.0	Juin 2020 
Refonte d'une application initialement développée pour Open-Sankoré.
Cette application permettait simplement d'afficher une image sur la page active de Open-Sankoré (seyes, mm, quadrillage...)

- Simplification  des algorithmes générant les feuilles
- Création d'onglets
- Possibilité de paramétrer
	* les couleurs de lignes
	* les hauteurs d'interligne
	* les maillages des réseaux pointés
- Ajout d'une fenêtre d'aperçu
- Ajout de l'image en fond de page
- Adaptation du fond de page (16/9 ou 4/3)

		value:15,
		min: 10,
		max: 60,
		step: 5,
		slide: function( event, ui ) {
				$( "#taille_carreau_seyes" ).val( ui.value );//Actualisation de la valeur affichée
			},
		change: function( event, ui ) {
				dessiner('seyes','aperçu'); // Actualisation de l'aperçu
			}
	});
